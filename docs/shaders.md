# Shader Catalog

Complete list of all available shaders organized by category.

## CRT/Retro Display Effects

Simulate vintage cathode-ray tube monitors with scanlines, phosphor masks, and screen curvature.

| Shader | Description |
|--------|-------------|
| `crt.glsl` | Full CRT simulation with scanlines, shadow masks, tube warping, and vignetting |
| `bettercrt.glsl` | Enhanced CRT effect with improved visuals |
| `in-game-crt.glsl` | Gaming-style CRT effect |
| `retro-terminal.glsl` | Classic green terminal aesthetic |
| `tft.glsl` | TFT/LCD display simulation |

## Space/Celestial Effects

Animated cosmic backgrounds and space-themed visuals.

| Shader | Description |
|--------|-------------|
| `starfield.glsl` | Moving white stars with parallax layers |
| `starfield-colors.glsl` | Colorful starfield variation |
| `galaxy.glsl` | Galaxy and nebula effects |
| `inside-the-matrix.glsl` | 3D Matrix-style digital rain with camera movement |
